    • the string in

Runfile

(run|runfile) filename+=Filename[,]

  • Type one of the following:
    • the string run
    • the string runfile
  • Type at least one Filename separated by comma


Filename

Filename is a regex rule, detecting anything matched by /[a-zA-Z0-9\.\/\~\[\]\'"_-]+/, such as:

  • ../../a/relative/path/to/a/.dotfile
  • 007
  • A/Relative/Path/To/A/.Dotfile
  • lucas


Identifier

Identifier is a regex rule, detecting anything matched by /[a-zA-Z0-9_-]+/, such as:

  • 007
  • 42
  • Lucas
  • avalidpythonidentifier_02342
